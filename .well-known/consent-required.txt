# Consent Requirements for eTMA Handler
# This file documents data handling and consent requirements

## Data Processed

This application processes:
- Student assignment submissions (.fhi files)
- Tutor feedback and grades
- Configuration preferences

## Consent Status

| Data Type | Consent Required | Basis |
|-----------|------------------|-------|
| Student submissions | Institutional | Educational legitimate interest |
| Tutor grades | Employment | Contractual obligation |
| Application settings | None | Functional necessity |

## Data Retention

- Assignment data: User-controlled (local storage)
- Backups: 7 days default (configurable)
- Soft-deleted items: 30 days

## Data Location

All data stored locally:
- Linux: ~/.local/share/etma_handler/
- macOS: ~/Library/Application Support/etma_handler/
- Windows: %APPDATA%\etma_handler\

## No External Transmission

This application:
- Does NOT transmit data to external servers
- Does NOT include telemetry
- Does NOT require network access for core functionality

## User Rights

Users may:
- Export all their data at any time
- Delete all local data
- Configure backup retention
- Disable all optional features
