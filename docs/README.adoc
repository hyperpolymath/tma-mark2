== Conventions

* AsciiDoc (`.adoc`) for long-form documents that benefit from a
  table of contents or structured cross-references.
* Markdown (`.md`) for shorter pieces and anything GitHub renders by
  default.
* File names are kebab-case under `docs/`. The top-level community
  files (README, CHANGELOG, CONTRIBUTING, …) keep their conventional
  UPPER_CASE names.
* Every file carries an SPDX-License-Identifier header.
