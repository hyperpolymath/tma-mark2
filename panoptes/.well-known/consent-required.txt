# Panoptes Consent Information

This software processes files on your local system.

## What Panoptes Does

1. READS files from directories you specify
2. ANALYZES content using local AI models (Ollama)
3. RENAMES files based on AI analysis
4. STORES metadata in a local SQLite database
5. LOGS all operations for undo capability

## What Panoptes Does NOT Do

1. Does NOT send data to external servers
2. Does NOT require internet after initial setup
3. Does NOT modify file contents (only names)
4. Does NOT delete files (only renames)
5. Does NOT operate without explicit user command

## Data Storage

- Database: ~/.local/share/panoptes/panoptes.db
- History: ~/.local/share/panoptes/history.jsonl
- Config: ~/.config/panoptes/config.json

## Privacy

All processing is local. No telemetry is collected.
The Ollama AI model runs entirely on your machine.

## Reversibility

All operations can be undone using:
  panoptes undo

Full history is maintained in the history file.

## Consent

By using Panoptes, you consent to:
- Local file system access (read/rename)
- Local database storage
- Local AI model inference

No external consent is required as no data leaves your machine.
