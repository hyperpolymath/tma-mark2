== What is Panoptes?

Panoptes is an AI-powered local file scanner that automatically:

* Analyzes images, documents, and media files
* Generates intelligent filenames based on content
* Applies tags for easy organization
* Runs entirely offline using local AI models

== Features

=== AI-Powered Analysis

Panoptes uses the Moondream vision model via Ollama to understand file contents:

* **Images**: Describes visual content and suggests meaningful names
* **PDFs**: Extracts text and summarizes document content
* **Audio**: Reads metadata and organizes music files
* **Video**: Cleans up video filenames
* **Code**: Identifies programming languages and project types
* **Archives**: Inspects contents without extraction

=== Privacy First

* All processing happens locally on your machine
* No data is sent to external services
* Works completely offline after initial setup

=== Reversible Operations

* Every file rename is logged
* Undo any operation at any time
* Full history tracking

== System Requirements

* **OS**: Linux, macOS, Windows
* **Memory**: 4GB RAM minimum (8GB recommended)
* **Disk**: ~2GB for Moondream model
* **Ollama**: Required for AI features
